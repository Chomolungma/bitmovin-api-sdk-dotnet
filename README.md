> Don't have an account yet? [Sign up for a free Bitmovin trial plan](https://dashboard.bitmovin.com/signup)!

For full documentation of all available API endpoints, see the [Bitmovin API reference](https://bitmovin.com/docs).

## Installation

### Nuget

Add this dependency to your project:

Package Manager
```shell
Install-Package Bitmovin.Api.Sdk -Version 1.33.0-alpha.0
```

.NET CLI

```shell
dotnet add package Bitmovin.Api.Sdk --version 1.33.0-alpha.0
```

## Initialization

```csharp
    using Bitmovin.Api;

    var bitmovinApi = BitmovinApi.Builder
        .WithApiKey(API_KEY)
        .WithLogger(new XunitLogger(output))
        .Build();  
```
